# Tutu AI prototype

Минимальный прототип: FastAPI, статическое окно чата, обязательный прокси,
OpenAI Responses API, `gpt-5.6-terra`, Tutu MCP и строгий JSON-ответ.

История последних 20 сообщений и параметры поездки сохраняются только в
`sessionStorage` текущей вкладки браузера. Серверной БД пока нет; закрытие
вкладки очищает этот контекст.

Модель возвращает `trip_update`, поэтому найденные в разговоре города, даты и
число пассажиров автоматически синхронизируются с редактируемой панелью.
Фронтенд безопасно отображает `**жирный текст**` и встраивает ссылки Tutu в
совпадающие фразы ответа без вставки непроверенного HTML.
Если модель всё же вернёт Markdown-ссылку, фронтенд покажет только её название
и сделает его кликабельным лишь для корректного адреса `tutu.ru` или его
поддомена. Повреждённые и сторонние URL остаются обычным текстом.

После получения полного JSON фронтенд быстро анимирует печать ответа. Длинные
сообщения выводятся увеличенными порциями и занимают не более нескольких
секунд; клик по печатающемуся сообщению показывает его целиком.

Бэкенд передаёт модели текущую дату, часовой пояс и точные даты ближайших
выходных. По умолчанию используется `APP_TIMEZONE=Europe/Moscow`.

Непустые параметры боковой панели считаются подтверждёнными. Ответ модели
сливается с ними: `null` не может стереть известную дату, город или число
пассажиров, а явное новое значение обновляет только соответствующее поле.

## Запуск

Команды выполняются из папки `project`:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r back\requirements.txt
Copy-Item .env.example .env
```

Заполните `OPENAI_API_KEY` и `PROXY` в `.env`. Пока нового `.env` нет,
прототип также может прочитать эти две переменные из родительского `.env` —
это сделано только для совместимости с пятью нетронутыми референсными файлами.

Проверка прокси без вызова OpenAI:

```powershell
python -m back.check_proxy
```

Smoke-проверка всей цепочки OpenAI + Tutu MCP + JSON Schema:

```powershell
python -m back.smoke_chat
```

Запуск приложения:

```powershell
python -m uvicorn back.main:app --reload
```

Откройте <http://127.0.0.1:8000>.

Прямого сетевого fallback нет: если `PROXY` отсутствует или его проверка
не проходит, запрос к OpenAI не выполняется.
